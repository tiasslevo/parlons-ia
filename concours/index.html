Concours ouvert

Crée une invitation de mariage avec l'IA

Pour le premier concours Parlons IA, l'objectif est simple : produire une carte d'invitation élégante, lisible et créative avec un outil de génération d'image.

Remplir le formulaire Ouvrir Tally
Prix 100$

Pour la meilleure création.

Deadline 1er juin

Clôture à 23h59.

Format Image

PNG, JPG ou WEBP.

Le défi

Créer une carte d'invitation de mariage

Utilise ChatGPT Image, Gemini ou un autre outil de génération d'image pour créer une invitation de mariage élégante. Le but n'est pas de partir d'un template tout fait, mais de montrer ce que tu arrives à obtenir avec un vrai travail de prompt.

Tu peux itérer, améliorer ton prompt, recadrer ou exporter ton image proprement. La création finale doit surtout rester cohérente, lisible et utilisable comme une vraie carte d'invitation.

À soumettre

  • Image finale de la carte.
  • Prompt utilisé.
  • Outil ou modèle utilisé.
  • Nom ou pseudo + contact.
  • Démarche optionnelle en quelques lignes.

Outils recommandés

ChatGPT Image et Gemini sont recommandés pour ce type de défi, surtout pour la gestion du texte dans l'image. Les autres générateurs restent autorisés si le rendu final respecte le cadre.
